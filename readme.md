# Packing Checklist

A tiny Jekyll checklist for packing.

View it here: [craigatallahfrost.com/packing](http://craigatallahfrost.com/packing/)

The checklist content lives in [_includes/checklist.md](_includes/checklist.md). The page wrapper is [index.md](index.md), and checked items are saved locally in your browser with [assets/checklist.js](assets/checklist.js).
