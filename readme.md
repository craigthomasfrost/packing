# Packing Checklist

This repo keeps a reusable master packing checklist as a GitHub issue template.

## How To Use It

1. Open the repo on GitHub.
2. Go to **Issues**.
3. Click **New issue**.
4. Choose **Trip Packing Checklist**.
5. Fill in the trip details, delete any modules that do not apply, and use the issue checkboxes while packing.

The issue template is the source of truth for the checklist so each trip gets its own clickable, editable packing list.
